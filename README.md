# Medical App Project

This is a Flutter-based medical application designed to connect patients with doctors, offering features such as specialty browsing, doctor listings, detailed doctor profiles, and chat functionality. The project leverages a modern technology stack including Flutter, Riverpod for state management, GoRouter for navigation, and Firebase for backend services.

## Features

*   User Authentication (Firebase Auth)
*   Doctor and Specialty Management (Firestore)
*   Chat Functionality
*   Multi-language Support (Arabic and English)
*   Responsive UI with Theming
*   **Project Security Gate (Password Protected)**

## Technology Stack

*   **Framework**: Flutter (Dart)
*   **State Management**: Riverpod
*   **Navigation**: GoRouter
*   **Backend**: Firebase (Authentication, Firestore, Storage, Messaging)

## Getting Started

To run this project locally, ensure you have Flutter installed. Then:

1.  Clone the repository.
2.  Navigate to the `medical_app` directory.
3.  Run `flutter pub get` to install dependencies.
4.  Set up your Firebase project and add your `google-services.json` (Android) or `GoogleService-Info.plist` (iOS) file to the respective platform folders. Uncomment the Firebase initialization in `lib/main.dart`.
5.  Run the app using `flutter run`.
